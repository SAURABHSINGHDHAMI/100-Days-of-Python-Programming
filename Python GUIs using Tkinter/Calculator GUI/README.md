# 🧮 Calculator GUI

This is a simple calculator GUI created using the Tkinter library in Python. The calculator provides basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features 🚀

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear (C)
- Equals (=)

## How to Use 📜

1. Run the Python script in your terminal or preferred Python environment.
   ```bash
   python calculator.py
   ```

2. The calculator GUI window will appear.

3. Use the numeric buttons (0-9) to input numbers.

4. Use the operation buttons (+, -, *, /) to perform calculations.

5. Press the "C" button to clear the display.

6. Press the "=" button to obtain the result of the operation.

## Example Usage 💡

1. To add two numbers:
   - Click on the numeric buttons to enter the first number.
   - Click on the "+" button.
   - Enter the second number.
   - Press "=" to get the result.

2. To clear the display:
   - Click on the "C" button.

3. Division with error handling:
   - When dividing by zero, the calculator displays an "Error" message.

## GUI Layout 🖼️

The GUI layout consists of numeric buttons, operation buttons, and a result label.

## Dependencies 🛠️

- Tkinter
